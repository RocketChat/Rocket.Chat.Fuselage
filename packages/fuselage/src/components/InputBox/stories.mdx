# InputBox

A decorated version of <LinkTo kind='Forms|InputControl' story='Default Story'>`InputControl`</LinkTo> with support for
addons.

Usually you'll perfer to use `-Input` (e.g. <LinkTo kind='Forms|TextInput' story='Default'>`TextInput`</LinkTo>)
components over this one because it works as a construction block for them.

<Preview>
  <Story name='Default'>
    <InputBox defaultValue='Value' onChange={action('change')} />
  </Story>
</Preview>

<Props of={InputBox} />

## With Addon

Any other component can be attached to a <LinkTo kind='Forms|InputBox' story='Default Story'>`InputBox`</LinkTo> as a
addon.

<Preview>
  <Story name='With Addon'>
    <InputBox addon={<Button small ghost>Button</Button>} />
  </Story>
</Preview>

If the addon component is a <LinkTo kind='Misc|Icon' story='Default'>`Icon`</LinkTo>, its size and color will be
adjusted to match the states of the <LinkTo kind='Forms|InputBox' story='Default Story'>`InputBox`</LinkTo>.

<Preview>
  <Story name='With Icon Addon'>
    <InputBox addon={<Icon name='send' />} />
  </Story>
</Preview>
