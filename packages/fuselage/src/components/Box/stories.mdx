# Box

A primitive component with normalized styles.

<Props of={Box} />

### `is` prop

The `is` prop allows `Box` to render any component capable of handling common DOM attributes, specially `className` and
`style`.

  <Story name='is prop'>
    <Box is={Button}>A Box rendering a Button</Box>
